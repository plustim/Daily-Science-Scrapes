# Live-Science-Scrapes

[Use this app](https://live-science-scraper.herokuapp.com/)  
This app allows the user to scrape the website [Live Science](https://www.faceplusplus.com/) for new articles about animals.  The article headlines are then saved to the database.  Users can write notes on these articles, which are then saved and viewable by any visitors.

### Using
- [Node.js](https://nodejs.org/en/)
- [Express.js](http://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)
- [Skeleton](http://getskeleton.com/)
- [jQuery](http://jquery.com/)
